# Normalización de Bases de Datos

Repositorio con los ejercicios del módulo de Normalización de Bases de Datos. Cada ejercicio parte de una tabla sin normalizar, la normaliza en formas normales (1NF, 2NF, 3NF), y documenta el diseño resultante con un diagrama Entidad-Relación de Chen y un diagrama de base de datos en notación de patas de gallo.

## Herramientas utilizadas

- Google Sheets / Excel — normalización de datos
- [diagrams.net](https://app.diagrams.net) — diagramas ER de Chen y de patas de gallo
- [Mermaid](https://mermaid.js.org/syntax/entityRelationshipDiagram.html) — diagramas embebidos directamente en los README
